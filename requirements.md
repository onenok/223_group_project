# CIS223 Web Application Development CA/ CIS226 Advanced Web Application Development

Group Project (50%)

Background

Your company is employed as the web application developer, and you are now requested to create a web-based application which help the online store to run their business and also add the data mining function for do shopping habit analysis through the data collected and stored from database.

The web application required to have the following function:

- User can register as member.
- Both user and anonymous user can browse the website.
- Anonymous user can register as a member.
- Registered user can add the selected items to the shopping cart.
- Registered user can checkout the cart and do payment (student do not need to really add the payment gateway, you can simply redirect to the invoice page that show the user information like name, phone number, address, list of product bought and other related information instead.)
- Database will store the shopping records, and data mining function should add to the web application to find out information like what is the hot product, complementary product, suggested list and etc.

Student should setup the server in the local machine and run the web on local machine only. Thus, student should install the web server, database server and also choose the framework. You can choose any platform and framework to create the website.

Student should use html, css, javascript or jQaury to design their own website.

Student are required to complete the website, also need to do a set of user manual to tell the client how to install all the components used and use screen capture to describe to flow of using the website.

Group Size : 3-5

Deliverables:

- Complete website

Zip the root folder and submit to moodle.

- Database

Export the database as .sql file and submit to moodle.

- User manual

Use a word document to make the user manual and submit to moodle.

Presentation:

- Presentation should be done by face to face mode or recordings.
- Each group member should participate in the presentation.
- Time limit of presentation is 15mins - 20mins.
- Presentation PowerPoint is required.
